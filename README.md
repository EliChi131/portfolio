# Portfolio

Starter files for a personal portfolio website.

## Project Structure

- index.html: Main page structure and content sections.
- css/styles.css: Site styling, layout, and responsive behavior.
- js/main.js: Small interactive features (mobile menu and footer year).
- assets/images: Place images for projects/profile here.
- assets/files: Place resume or downloadable files here.

## Run Locally

1. Open the `portfolio` folder in VS Code.
2. Open `index.html` in your browser.
3. Optional: use a live server extension for auto-reload while editing.

## Customize Next

1. Update your name and intro text in `index.html`.
2. Replace project placeholders with your real projects and links.
3. Add your email, LinkedIn, and GitHub links.
4. Adjust colors and spacing in `css/styles.css`.
